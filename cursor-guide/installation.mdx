## Installation Steps

<Tabs>
  <Tab title="macOS">
    ### macOS Installation
    
    <Steps>
      <Step title="Download">
        Go to [cursor.sh](https://cursor.sh) and download the `.dmg` file
      </Step>
      <Step title="Open DMG">
        Double-click the downloaded file
      </Step>
      <Step title="Drag to Applications">
        Drag Cursor to your Applications folder
      </Step>
      <Step title="Launch">
        Open Cursor from Applications or Spotlight
      </Step>
      <Step title="Security">
        If macOS blocks it, go to System Preferences > Security & Privacy and click "Open Anyway"
      </Step>
    </Steps>
    
    <Tip>
      **Tip:** Add Cursor to your Dock for easy access
    </Tip>
  </Tab>
  
  <Tab title="Windows">
    ### Windows Installation
    
    <Steps>
      <Step title="Download">
        Download the Windows installer from [cursor.sh](https://cursor.sh)
      </Step>
      <Step title="Run Installer">
        Double-click the `.exe` file
      </Step>
      <Step title="Follow Wizard">
        Accept the license and follow prompts
      </Step>
      <Step title="Launch">
        Cursor will launch automatically after installation
      </Step>
    </Steps>
    
    <Info>
      Cursor installs like any Windows application. No special setup needed.
    </Info>
  </Tab>
  
  <Tab title="Linux">
    ### Linux Installation
    
    <Steps>
      <Step title="Download Package">
        Download `.AppImage`, `.deb`, or `.rpm` from [cursor.sh](https://cursor.sh)
      </Step>
      <Step title="AppImage (All Distros)">
        ```bash
        chmod +x cursor-*.AppImage
        ./cursor-*.AppImage
        ```
      </Step>
      <Step title="Debian/Ubuntu (.deb)">
        ```bash
        sudo dpkg -i cursor_*.deb
        sudo apt-get install -f
        ```
      </Step>
      <Step title="Fedora/RHEL (.rpm)">
        ```bash
        sudo rpm -i cursor-*.rpm
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

## First Launch Setup

When you first open Cursor, you'll go through a quick setup:

<Steps>
  <Step title="Welcome Screen">
    You'll see Cursor's welcome screen. Click "Get Started"
  </Step>
  <Step title="Choose Your AI Model">
    Select your preferred AI model:
    - **GPT-4** (recommended for best results)
    - **GPT-3.5** (faster, good for simple tasks)
    - **Claude** (excellent for analysis)
    
    <Tip>
      Don't worry - you can change this later or switch per-request
    </Tip>
  </Step>
  <Step title="Sign In or Try Free">
    - **Sign up** for Cursor account (free tier available)
    - Or **use your own API key** (OpenAI, Anthropic)
    - Or **try without signing in** (limited features)
  </Step>
  <Step title="Optional: Import Settings">
    If you've used VS Code before, you can import your settings
    
    Otherwise, skip this step
  </Step>
</Steps>

## Configuring Cursor for Obsidian

Let's optimize Cursor for working with your Obsidian vault:

### Step 1: Basic Settings

<Steps>
  <Step title="Open Settings">
    Click the gear icon ‚öôÔ∏è or press `Cmd/Ctrl + ,`
  </Step>
  <Step title="Configure Editor">
    - **Font Size:** 14-16 for comfortable reading
    - **Word Wrap:** Enable (for long lines in notes)
    - **Auto Save:** Set to "afterDelay"
  </Step>
  <Step title="File Association">
    Tell Cursor to treat `.md` and `.mdx` files as markdown:
    - Search for "file associations" in settings
    - Add: `*.md` ‚Üí `markdown`
    - Add: `*.canvas` ‚Üí `json`
  </Step>
</Steps>

### Step 2: AI Settings

<Steps>
  <Step title="Open Cursor Settings">
    Go to Cursor Settings (separate from VS Code settings)
  </Step>
  <Step title="Choose AI Model">
    Under "Models":
    - **Main model:** GPT-4 or Claude 3.5 Sonnet (best quality)
    - **Fast model:** GPT-3.5 Turbo (for quick tasks)
  </Step>
  <Step title="Context Settings">
    - **Include .gitignore files:** Off (to read all notes)
    - **Max context:** 200K tokens (for large vaults)
  </Step>
</Steps>

### Step 3: API Keys (Optional)

If you want to use your own API keys instead of Cursor's credits:

<Accordion title="Setting up OpenAI API Key">
  1. Go to [platform.openai.com](https://platform.openai.com)
  2. Create an account and add payment method
  3. Generate an API key
  4. In Cursor: Settings ‚Üí Cursor Settings ‚Üí API Keys
  5. Paste your OpenAI key
  
  **Cost:** ~$0.01-0.10 per request depending on usage
</Accordion>

<Accordion title="Setting up Anthropic API Key">
  1. Go to [console.anthropic.com](https://console.anthropic.com)
  2. Create an account and add credits
  3. Generate an API key
  4. In Cursor: Settings ‚Üí Cursor Settings ‚Üí API Keys
  5. Paste your Anthropic key
  
  **Cost:** Similar to OpenAI, pay-as-you-go
</Accordion>

## Installing Useful Extensions

Cursor supports VS Code extensions. Here are some useful ones:

<Steps>

    Click the Extensions icon on the left sidebar (or Cmd/Ctrl + Shift + X)
  </Step>

    - **Markdown Preview Enhanced** - Better preview
    - **Paste Image** - Easy image insertion

## Troubleshooting

<AccordionGroup>
  <Accordion title="Cursor won't launch on macOS">
    **Solution:** 
    - Check System Preferences ‚Üí Security & Privacy
    - Click "Open Anyway"
    - Make sure macOS is 10.15 or newer
  </Accordion>
  
  <Accordion title="AI not responding">
    **Solution:**
    - Check your internet connection
    - Verify you're signed in or have API keys set
    - Check if you've exceeded free tier limits
    - Try switching to a different model
  </Accordion>
  
  <Accordion title="Can't open Obsidian vault folder">
    **Solution:**
    - Use File ‚Üí Open Folder
    - Navigate to your vault folder
    - Make sure you have read/write permissions
    - On Mac, you may need to grant Cursor folder access
  </Accordion>
  
  <Accordion title="Slow performance">
    **Solution:**
    - Close unnecessary tabs
    - Reduce max context in settings
    - Disable resource-heavy extensions
    - Consider using a faster AI model for simple tasks
  </Accordion>
</AccordionGroup>
