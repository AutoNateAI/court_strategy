## Essential Core Plugins

Enable these built-in plugins first:

### Daily Notes
**Why:** Perfect for journaling and daily planning

<Accordion title="How to set up Daily Notes">
  1. Go to Settings ‚Üí Core Plugins
  2. Enable "Daily notes"
  3. Click the gear icon next to it
  4. Configure:
     - Date format: `YYYY-MM-DD`
     - New file location: `Daily Notes/`
     - Template: Choose from Templates folder
  
  **Usage:** Click calendar icon or use `Cmd/Ctrl + Shift + D`
</Accordion>

### Templates
**Why:** Create consistent note structures

<Accordion title="How to set up Templates">
  1. Enable "Templates" core plugin
  2. Create a "Templates" folder in your vault
  3. Create template notes (e.g., "Meeting Template.md")
  4. In settings, set template folder location
  
  **Usage:** `Cmd/Ctrl + T` to insert template
</Accordion>

### Backlinks
**Why:** See which notes link to your current note

<Accordion title="Enable Backlinks">
  1. Enable "Backlinks" in core plugins
  2. View in right sidebar
  3. Discover connections you didn't know existed!
</Accordion>

### Graph View
**Why:** Visualize your knowledge network

<Accordion title="Configure Graph View">
  1. Enable "Graph view"
  2. Click the graph icon to view
  3. Use filters to focus on specific notes
  4. Watch your knowledge network grow!
</Accordion>

## Must-Have Community Plugins

### 1. Dataview üìä
**Create dynamic lists and tables from your notes**

<Accordion title="What is Dataview?">
  Dataview lets you query your notes like a database.
  
  **Example use cases:**
  - List all tasks across your vault
  - Show upcoming deadlines
  - Track project status
  - Generate reports
  
  **Install:** Settings ‚Üí Community plugins ‚Üí Browse ‚Üí Search "Dataview"
</Accordion>

**Example query:**
```dataview
TABLE status, deadline
FROM "Projects"
WHERE status = "in-progress"
SORT deadline ASC
```

### 2. Calendar üìÖ
**Visual calendar interface for daily notes**

<Accordion title="Why Calendar is essential">
  - See your daily notes on a calendar
  - Quickly navigate between dates
  - Spot patterns in your journaling
  - Perfect for families tracking events
  
  **Install:** Search "Calendar" in community plugins
  **Usage:** Shows in right sidebar
</Accordion>

### 3. Templater ‚öôÔ∏è
**Advanced template functionality**

<Accordion title="Templater vs Templates">
  While the core Templates plugin is good, Templater adds:
  - Dynamic dates and times
  - User input prompts
  - JavaScript execution
  - File operations
  
  **Install:** Search "Templater" in community plugins
  
  **Example template:**
  ```
  # Meeting with <% tp.file.title %>
  Date: <% tp.date.now("YYYY-MM-DD") %>
  Attendees: <% tp.system.prompt("Who attended?") %>
  ```
</Accordion>

### 4. Kanban üìã
**Visual task management boards**

<Accordion title="Kanban for project management">
  Create Trello-style boards right in Obsidian.
  
  **Great for:**
  - Project management
  - Tracking legal case tasks
  - Family to-do lists
  - Research workflow stages
  
  **Install:** Search "Kanban" in community plugins
  **Usage:** Create note with `.kanban` extension
</Accordion>

### 5. Excalidraw ‚úèÔ∏è
**Draw diagrams and illustrations**

<Accordion title="Visual thinking with Excalidraw">
  Draw diagrams, sketch ideas, annotate images.
  
  **Use cases:**
  - Family trees
  - Timeline diagrams
  - Process flows
  - Mind maps
  
  **Install:** Search "Excalidraw" in community plugins
</Accordion>

## Plugin Safety

<Warning>
  **Security considerations:**
  - Plugins can access all your vault files
  - Only install plugins with good reviews
  - Check last update date (avoid abandoned plugins)
  - Read permissions carefully
</Warning>

<Tip>
  Start with popular, well-maintained plugins. The community vets them thoroughly.
</Tip>

3. Test performance

## Plugin Troubleshooting

<AccordionGroup>
  <Accordion title="Plugin won't enable">
    **Solution:** 
    - Check if community plugins are enabled
    - Try uninstalling and reinstalling
    - Check plugin's GitHub for known issues
  </Accordion>
  <Accordion title="Obsidian is slow">
    **Solution:**
    - Disable plugins one by one to find the culprit
    - Common culprits: Dataview on large vaults, heavy themes
    - Update all plugins to latest versions
  </Accordion>
  <Accordion title="Plugin conflicts">
    **Solution:**
    - Some plugins don't work well together
    - Check plugin documentation
    - Try disabling one of the conflicting plugins
  </Accordion>
</AccordionGroup>
