## Getting Started

<AccordionGroup>

    Our workshops are specifically designed for non-technical users.
  </Accordion>
  
  <Accordion title="Which tool should I learn first?">
    Start with **Obsidian**:
    1. Install Obsidian and create your vault
    2. Get comfortable taking notes and linking
    3. Then add Cursor to supercharge your workflow
    
    You can get value from Obsidian immediately, then add AI power when you're ready.
  </Accordion>
  
  <Accordion title="How long does it take to learn?">
    **Obsidian basics:** 2-4 hours to feel comfortable
    **Cursor basics:** 1-2 hours to start being productive
    **Mastery:** Ongoing - you'll keep discovering new techniques
    
    Most people see value in the first week of use.
  </Accordion>
  
  <Accordion title="Is this really free?">
    **Obsidian:** Free for personal use. Paid options:
    - Sync: $10/month (optional)
    - Publish: $8/month (optional)
    
    **Cursor:** Free tier available. Pro: $20/month
    
    **Our workshops:** Some free, some paid - check schedule
    
    You can get started completely free!
  </Accordion>
</AccordionGroup>

## Obsidian Questions

<AccordionGroup>
  <Accordion title="Where are my notes stored?">
    Your notes are stored as plain text files (`.md`) on your computer, in a folder you choose (your "vault").
    
    **Benefits:**
    - You own your data completely
    - Works offline
    - Can back up like any other files
    - No vendor lock-in
    - Future-proof format
  </Accordion>
  
  <Accordion title="Can I use Obsidian on my phone?">
    Yes! Obsidian has iOS and Android apps. You can:
    - View and edit notes on mobile
    - Sync using Obsidian Sync ($10/month)
    - Or use iCloud/Dropbox for free sync
    
    [Installation guide →](/obsidian-guide/installation)
  </Accordion>
  
  <Accordion title="What's the difference between Obsidian and Notion/Evernote?">
    **Key differences:**
    
    **Obsidian:**
    - Files stored locally
    - Plain text markdown
    - Powerful linking
    - Privacy-focused
    - Works offline
    
    **Notion/Evernote:**
    - Cloud-based
    - Proprietary format
    - Easier collaboration
    - Requires internet
    - Monthly fees typically required
    
    Choose Obsidian if you want control and privacy.
  </Accordion>
  
  <Accordion title="Can I import from other apps?">
    Yes! You can import from:
    - **Notion:** Export as markdown, import to Obsidian
    - **Evernote:** Use export tools
    - **Apple Notes:** Copy/paste or use scripts
    - **Google Docs:** Convert to markdown
    - **Word:** Save as plain text or markdown
    
    Most content converts well, though formatting may need adjustment.
  </Accordion>
  
  <Accordion title="How do I backup my vault?">
    Multiple options:
    1. **Obsidian Sync:** Automatic, paid ($10/month)
    2. **Cloud storage:** Put vault in Dropbox/iCloud/Google Drive
    3. **Git:** Version control for tech users
    4. **Manual:** Copy vault folder regularly
    
    [Learn more →](/obsidian-guide/vault-setup#backup-strategy)
  </Accordion>
  
  <Accordion title="What happens if Obsidian goes away?">
    Your notes will still work! They're just plain text files. You can:
    - Open them in any text editor
    - Use another markdown app
    - Keep using them forever
    
    This is the power of plain text - it's future-proof.
  </Accordion>
</AccordionGroup>

## Cursor Questions

<AccordionGroup>
  <Accordion title="What is Cursor and why do I need it?">
    **Cursor** is an AI-powered editor that helps you work with your Obsidian notes:
    
    **What it does:**
    - Generate content automatically
    - Create visual diagrams (canvases)
    - Organize and rename files
    - Summarize and analyze notes
    - Automate repetitive tasks
    
    **Why you need it:**
    Makes working with large vaults 10x faster and enables things that would be tedious manually.
  </Accordion>
  
  <Accordion title="Will Cursor change my notes without asking?">
    No! Cursor will:
    - Show you what it plans to do
    - Wait for your approval
    - Let you undo changes
    
    You're always in control. Plus, if you back up regularly, you can always revert.
  </Accordion>
  
  <Accordion title="Is my data sent to AI companies?">
    **What gets sent:**
    - Your prompts (requests)
    - Files you specifically @mention
    - Open files (for context)
    
    **What doesn't:**
    - Your entire vault automatically
    - Files you haven't mentioned
    
    **Privacy options:**
    - Use your own API keys for more control
    - Use local AI models (advanced)
    - Be selective with what you @mention
  </Accordion>
  
  <Accordion title="How much does Cursor cost?">
    **Free Tier:**
    - 2,000 completions/month
    - 50 premium requests
    - Good for trying it out
    
    **Pro ($20/month):**
    - Unlimited completions
    - Unlimited premium requests
    - Access to best models
    - Recommended for regular use
    
    **Bring Your Own Key:**
    - Use OpenAI/Anthropic API directly
    - Pay-as-you-go (usually $10-30/month)
    - Most control
    
    [Learn more →](/cursor-guide/installation#pricing-options)
  </Accordion>
  
  <Accordion title="Can I use Cursor without Obsidian?">
    Yes! Cursor is useful for any text-based work:
    - Writing
    - Coding
    - Documentation
    - Research notes
    
    But it's especially powerful with Obsidian because of the structured note system.
  </Accordion>
</AccordionGroup>

## Technical Questions

<AccordionGroup>
  <Accordion title="What's a vault?">
    A **vault** is just a folder on your computer containing:
    - Your notes (`.md` files)
    - Attachments (images, PDFs)
    - Configuration
    
    That's it! Nothing magical - just organized files.
  </Accordion>
  
  <Accordion title="What's markdown?">
    **Markdown** is a simple way to format text:
    - `# Heading` creates a heading
    - `**bold**` makes text bold
    - `*italic*` makes text italic
    - `[[link]]` creates a link
    
    It's easy to learn and works in many apps.
  </Accordion>
  
  <Accordion title="What are plugins?">
    **Plugins** add features to Obsidian:
    - Created by Obsidian team (core plugins)
    - Created by community (community plugins)
    - Extend functionality
    - Optional - use what you need
    
    [Plugin guide →](/obsidian-guide/plugins)
  </Accordion>
  
  <Accordion title="What's a canvas?">
    **Canvas** is Obsidian's visual workspace:
    - Arrange notes spatially
    - Draw connections
    - Create mind maps
    - Plan projects visually
    
    Think: infinite whiteboard for your notes.
    
    [Canvas guide →](/cursor-guide/canvas-creation)
  </Accordion>
  
  <Accordion title="What file formats does Obsidian support?">
    **Native:**
    - Markdown (`.md`)
    - Canvas (`.canvas`)
    
    **Attachments:**
    - Images (PNG, JPG, GIF, etc.)
    - PDFs
    - Audio files
    - Video files
    - Any file type (stored, not edited)
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Obsidian is running slowly">
    **Common causes:**
    - Very large vault (10,000+ notes)
    - Many plugins enabled
    - Large media files
    - Graph view open on huge vault
    
    **Solutions:**
    - Disable unused plugins
    - Split into multiple vaults
    - Optimize media files
    - Close graph view when not needed
  </Accordion>
  
  <Accordion title="My links are broken">
    **Common causes:**
    - Renamed files
    - Moved files
    - Changed folder structure
    
    **Solutions:**
    - Use Obsidian's "Find unlinked mentions"
    - Enable "Automatically update internal links" in settings
    - Use "Check for broken links" in command palette
  </Accordion>
  
  <Accordion title="Cursor isn't responding">
    **Check:**
    - Internet connection (AI needs it)
    - API key/credits remaining
    - Cursor is up to date
    
    **Try:**
    - Restart Cursor
    - Try a different AI model
    - Check Cursor status page
  </Accordion>
  
  <Accordion title="Sync isn't working">
    **For Obsidian Sync:**
    - Check subscription active
    - Verify internet connection
    - Check sync settings
    
    **For cloud sync (iCloud/Dropbox):**
    - Ensure vault is in synced folder
    - Wait for sync to complete
    - Check for conflicts
    - Avoid working on multiple devices simultaneously
  </Accordion>
  
  <Accordion title="I can't find my vault">
    **Find it:**
    - In Obsidian: Settings → About → Show vault folder
    - Search your computer for `.obsidian` folder
    - Check Documents folder (common location)
    
    **Prevention:**
    - Note vault location when creating
    - Keep vaults in consistent location
    - Back up regularly
  </Accordion>
</AccordionGroup>
