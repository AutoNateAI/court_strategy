## Cursor Prompts Library

### Ready-to-Use Prompts

Save these prompts for common tasks:

<AccordionGroup>
  <Accordion title="Timeline Generation">
    ```
    @[folder]/ Read all notes in this folder and create a timeline document.
    
    Format:
    # Timeline
    
    ## YYYY-MM-DD - Event
    **What happened:** Description
    **Source:** [[Note]]
    
    Sort chronologically, earliest first.
    ```
  </Accordion>
  
  <Accordion title="Comparison Table">
    ```
    Create a comparison table from these notes:
    @note1.md
    @note2.md
    @note3.md
    
    Include columns: Name, [criteria1], [criteria2], [criteria3], Pros, Cons
    Format as markdown table
    ```
  </Accordion>
  
  <Accordion title="Index Creation">
    ```
    @[folder]/ Create an index note called "[Name]-Index.md"
    
    List all notes in this folder organized by:
    - Topic/category
    - With brief description
    - Link to each note
    
    Format as nested bullet list
    ```
  </Accordion>
  
  <Accordion title="Summary Generation">
    ```
    @[file].md Create a summary of this note that:
    - Captures 5 main points
    - Is 1 paragraph
    - Uses professional tone
    - Includes links to any mentioned notes
    ```
  </Accordion>
</AccordionGroup>
